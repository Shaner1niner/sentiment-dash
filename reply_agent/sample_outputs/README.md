# Sample outputs

Run `python scripts\smoke_seta_reply_agent.py` after installation to generate and validate structured draft replies from the sample inputs.

This folder is intentionally left light so generated outputs do not clutter Git. If desired, write outputs here with:

```bat
python scripts\draft_seta_social_reply.py --platform x --comment "Why is $BTC ranked high?" --output-json reply_agent\sample_outputs\btc_example.json
```
